Implantação do Sistema de Planos, Créditos e Integração com Kiwify
Objetivo

Implementar um sistema de créditos integrado aos planos da plataforma, onde o usuário utiliza créditos para realizar exportações (downloads) e a compra dos planos é realizada pela Kiwify.

Regras Gerais
Plano Free

Ao criar uma conta:

O usuário recebe direito a 1 download gratuito.
Após utilizar esse download:
Continua podendo criar projetos.
Continua podendo editar projetos.
Continua podendo salvar projetos.
Não poderá exportar novamente sem possuir créditos.

Cada exportação deve consumir:

40 créditos.

Primeira Compra

Todo usuário possui um campo:

first_purchase = false

Enquanto este campo for false:

Mostrar os preços promocionais.

Após qualquer pagamento aprovado:

first_purchase = true

A partir desse momento:

Mostrar apenas os preços normais.

Controle de Créditos

Criar um campo chamado:

credits

Sempre que ocorrer uma exportação:

Verificar:

credits >= 40

Se verdadeiro:

Permitir exportação.
Subtrair 40 créditos.

Se falso:

Bloquear exportação.
Abrir tela de Upgrade.
Controle do Plano

Criar campo:

plan

Valores possíveis:

FREE

EASY

MEDIUM

PREMIUM

Este campo define quais modelos o usuário pode acessar.

Integração com Kiwify

Criar endpoint:

POST

/api/webhook/kiwify

Esse endpoint deverá:

Receber o webhook.
Validar a requisição.
Confirmar que o pagamento foi aprovado.
Identificar o usuário.
Identificar qual produto foi comprado.
Adicionar créditos.
Atualizar o plano.
Marcar:

first_purchase = true

Registrar a compra.
Registrar o histórico de créditos.
Retornar HTTP 200.
